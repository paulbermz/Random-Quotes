Buttons
  • New Quote: Generate a new random quote.
  • Prev Quote: View the previously generated quote.
  • Tweet Quote: Share the current quote on Twitter.
  • Voice Command: Allow user to use voice command "READ", "SPEAK", "BASAHABAY", and "BASAHINMO" for reading the displayed quote, "NEXT", "PROCEED", and "SUNOD" to generate a new quote, and "PREVIOUS", "PREV", and "BALIK" to view the previous quote.
  • Voice Button: The speech synthesis will read the current quote aloud.
  • Copy Quote: Copy the current quote to clipboard.

About
This system, crafted with a blend of dedication and passion for web development, showcases various features that enhance user interaction and accessibility. As a 2nd-year BSIT student aiming to become a full-stack web developer, I created this project to practice and improve my skills. The system offers a wide array of quote categories that users can explore, with additional functionalities such as voice synthesis for reading quotes aloud and easy social media sharing for spreading inspiration. The navigation is designed to be intuitive, ensuring that users can easily move between sections like Home, Settings, Contact, and About. The responsive design and the incorporation of dark mode and state persistence further enrich the user experience, making the application flexible and enjoyable to use across different devices. As I continue my journey in learning and mastering full-stack development, this project reflects my current understanding and growing expertise. It represents not just a functional tool, but a stepping stone in my ongoing education, where each feature I've implemented has taught me valuable lessons in user interface design, accessibility, and efficient coding practices. I'm excited to continue building on this foundation as I work towards becoming a proficient full-stack web developer.
